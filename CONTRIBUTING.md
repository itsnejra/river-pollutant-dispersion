# Contributing

1. Fork the repo and create a feature branch from `main`.
2. Install dev dependencies: `uv sync --group dev`.
3. Run the linter before committing: `ruff check src/ main.py`.
4. Open a pull request — please describe *what* changes and *why*.

## Running tests

There are currently no automated tests (the simulation is the validation).
A smoke-test script is planned for a future release.
